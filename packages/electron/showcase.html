数据流

从用户输入到 AI 响应
完整数据链路

消息从 ChatInput 发出，经 WebSocket 到达后端，AI 流式响应原路返回，全程实时无阻塞。

1

用户输入

ChatInput 捕获用户消息，调用 onSend 回调

sendMessage(content: string)
→ setMessages([...prev, userMsg])
→ wsService.sendMessage(content)
2

WebSocket 传输

WebSocketService 将消息序列化为 JSON 发送至服务端

{ "type": "message",
"content": "..." }
3

AI 流式响应

服务端通过 gRPC 调用 AI 引擎，逐 chunk 推送

case 'text':
streamingContentRef.current += event.content
case 'tool_call':
setPendingToolCall(event.toolCall)
4

UI 实时更新

React 状态驱动渲染，流式文本与工具卡片即时呈现

case 'done':
setMessages(prev => [...prev, {
role: 'assistant',
content: fullText
}])
